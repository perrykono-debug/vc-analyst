# TOOLS.md — 投资分析师专用工具备注

---

## 核心工具

### `online-search`
**必须优先使用**。搜索项目信息、创始人背景、行业动态、融资新闻。

使用方式：
```
online-search 搜索内容
```

### `neodata-financial-search`
金融数据搜索（A 股/港股/美股）。如目标已上市或有关联上市公司，使用此工具。

### `news-summary`
汇总近期新闻。分析项目风险、行业动态时使用。

---

## 辅助工具

### `pdf`
读取 PDF 文件（如用户上传 BP、行业报告）。

### `docx` / `xlsx` / `pptx`
导出分析报告（如用户需要）。

### `tencent-docs`
保存到腾讯文档（如用户需要在线协作）。

---

## 工具使用原则

1. **搜索优先**：禁止仅凭训练数据回答，必须先搜索
2. **多源验证**：关键信息至少 2 个来源验证
3. **标注来源**：每条关键数据注明搜索来源
4. **区分已知/推测**：无法确认的数据标注「估算」或「未知」

---

## 搜索策略

### 项目分析搜索清单

1. `[项目名] + 公司 + 简介 + 融资`
2. `[创始人姓名] + 背景 + 经历`
3. `[项目名] + 创始人 + 访谈 + 演讲`
4. `[行业] + 市场规模 + 增速 + 报告`
5. `[项目名] + 竞争对手 + 对比`
6. `[项目名] + 风险 + 诉讼 + 处罚`

### 创始人分析搜索清单

1. `[创始人] + 教育背景`
2. `[创始人] + 工作经历`
3. `[创始人] + 创业经历`
4. `[创始人] + 访谈 + 观点`
5. `[创始人] + 股权 + 关联公司`

---

## 禁止行为

- ❌ 不搜索直接回答
- ❌ 仅用一个来源
- ❌ 不标注数据来源
- ❌ 将推测当作事实
